# Курсовой проект: Веб-приложение для помощи в трудоустройстве

Это проект веб-сайта, который помогает соискателям находить вакансии, а работодателям — размещать предложения о работе и просматривать резюме. Проект написан на Python с использованием фреймворка Django, а в качестве базы данных используется SQLite.

## Что внутри проекта
* Система регистрации с разделением ролей: можно зарегистрироваться как "Соискатель" или как "Работодатель".
* Личный кабинет соискателя, где можно создать резюме и откликаться на вакансии.
* Личный кабинет работодателя для публикации вакансий и просмотра откликов.
* Встроенная админка Django для управления пользователями и модерации вакансий.

## Как запустить проект на компьютере

Чтобы запустить сайт локально, выполните три простых шага в терминале:

1. Установите необходимые библиотеки (важно, чтобы у вас уже был установлен Python):
pip install -r requirements.txt

2. Перейдите в папку, где лежит файл manage.py:
cd название_твоей_папки
(В этом проекте manage.py находится в корневой папке master)

3. Запустите сервер Django:
python manage.py runserver

## Как открыть сайт в браузере

После запуска третьей команды в терминале, в самом конце появится ссылка на локальный сервер. Обычно она выглядит так:

http://127.0.0.1:8000/

Просто нажмите или скопируйте и вставьте в строку браузера. Сайт откроется. Чтобы остановить работу сервера, нажмите в терминале комбинацию клавиш Ctrl + C.
